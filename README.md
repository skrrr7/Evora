# Evora

Evora is a simple, responsive financial tracking website built using the MERN stack. It helps users log session-based balances, track net profit/loss, and visualize performance trends and ROI over time.

## Features

* **Session Logging:** Keep track of financial entries by individual sessions.
* **Automated Calculations:** Instantly view net profit/loss and Return on Investment (ROI).
* **Performance Analytics:** Visual charts to easily spot trends and financial performance.
* **Responsive Design:** Works smoothly across desktop and mobile browsers.
